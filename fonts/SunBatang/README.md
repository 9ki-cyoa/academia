# SunBatang Font Files

이 폴더는 `SunBatang` TTF 폰트를 보관하기 위한 폴더입니다.

## Files

- `SunBatang-Light.ttf`
- `SunBatang-Medium.ttf`
- `SunBatang-Bold.ttf`

## Usage Note

- 사용자가 제공한 정보 기준으로 이 폰트는 비상업용 사용을 전제로 보관합니다.
- 상업용 프로젝트, 유료 배포물, 광고, 후원/판매가 포함된 콘텐츠에 사용하기 전에는 반드시 원 배포처의 라이선스를 다시 확인해야 합니다.
- 폰트 파일 자체를 재판매하거나 별도 상품처럼 배포하지 마세요.
- 실제 웹 페이지에서 사용할 때는 TTF 용량이 크므로 WOFF2 변환본을 별도로 만들어 쓰는 것을 권장합니다.
